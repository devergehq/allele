## Architecture patterns introduced

All six patterns are documented in `docs/architecture.md`.

1. **Command pattern** — `PendingAction` enum defers side-effectful work out of the GPUI render tree; 8 family sub-enums (`SessionAction`, `ArchiveAction`, `DrawerAction`, `SidebarAction`, `ProjectAction`, `SettingsAction`, `BrowserAction`, `OverlayAction`) group related variants. `dispatch_pending_action` fans out to per-family handlers.

2. **Adapter pattern** — Three traits (`CloneBackend`, `BrowserIntegration`, `SystemShell`) abstract platform operations. macOS impl delegates to existing modules; Linux impl ships graceful no-ops (`FullCopyCloneBackend`, `UnsupportedBrowser`, `PortableSystemShell`). `Platform` bundles all three behind `Arc<dyn Trait>`.

3. **Repository pattern** — `SettingsRepository` and `StateRepository` traits own all persistence I/O. Production impls read/write JSON; test fakes hold data in `Arc<Mutex<T>>`. Wired via `Repositories` bundle stored on `AppState`.

4. **Sub-struct composition** — `AppState` god-object decomposed into 5 cohesive bundles: `SidebarState`, `RightPanelState`, `DrawerState`, `EditorState`, `ConfirmationState`. Fields that belong together are colocated and can be passed as a unit.

5. **Typed errors + structured logging** — `AlleleError` (9 variants via `thiserror`) replaces ad-hoc `String` errors at `clone::*` and `git::*` call boundaries. `tracing` macros replace `println!`/`eprintln!`. `init_tracing()` wires `EnvFilter` at startup.

6. **Dirty-flag persistence coordinator** — `state_dirty` / `settings_dirty` flags on `AppState` are set by any mutation; a single coalesced write at the GPUI render tick calls `repos.state.save()` / `repos.settings.save()`. No scattered `write_settings()` call sites.

## Notable bugs fixed during the session

- **macOS BSD sed back-references** — `\1` doesn't work in BSD sed; all complex replacements switched to `perl -i -pe`.
- **PendingAction double-parentheses** — mechanical sed introduced `SessionAction::Foo((arg))`; fixed with a perl strip pass.
- **Tuple destructuring damage** — `Some((cursor, idx, buf))` became `Some(cursor, idx, buf)` during the strip; targeted perl repair.
- **`*skip_refocus = true` in family handlers** — parameter is `&mut bool`, requires deref on assignment.
- **`render_editor_context_menu` None guard** — was called only when `context_menu.is_some()` at the call site but used `.unwrap()` internally; refactored to handle `None` with early return inside the function, caller gate removed.
